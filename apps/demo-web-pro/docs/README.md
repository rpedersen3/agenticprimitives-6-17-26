# demo-web-pro docs

`demo-web-pro` is now the Treasury Service Agent demo from `specs/211-treasury-service-agent-demo.md`.

The old multi-sig capability gallery docs were removed because the app is no longer a set of independent flows. The demo now follows one story:

> Two people claim passkey-controlled Person Smart Agents, create an Organization Smart Agent, create a Treasury Service Agent, and progressively move authority from bootstrap ownership into agent-to-agent stewardship.

Start here:

- [`treasury-service-agent/guide.md`](treasury-service-agent/guide.md) — current app flow, web/a2a/MCP boundaries, and interaction diagrams.
